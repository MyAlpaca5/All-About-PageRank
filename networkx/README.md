# NetworkX PageRank
This module is used to verify the correctness of other solutions.

## Prerequisite
- Python 3.11

## Verification
- (Optionally) set up python virtual environment
  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  python3 -m pip install -r requirements.txt
  ```
- run networkx
  ```bash
  python3 pagerank.py
  ```
